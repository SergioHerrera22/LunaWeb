# Luna Metal Industrial - Landing Empresarial

Landing web empresarial desarrollada con React, TypeScript, Vite y Tailwind CSS para una metalurgica industrial enfocada en servicios para mineras.

## Stack

- React 18
- TypeScript
- Vite 5
- Tailwind CSS 4 (plugin oficial para Vite)
- Lucide React (iconografia)

## Caracteristicas incluidas

- Hero de alto impacto comercial con propuesta de valor minera.
- Seccion de servicios industriales (fabricacion, mantenimiento y montaje).
- Bloque de credenciales/metricas para respaldo comercial.
- Galeria visual de trabajos asociados al rubro metalurgico-minero.
- CTA final de ventas para captar consultas.
- Diseno responsive para desktop y mobile.
- Animaciones suaves y estilo visual industrial premium.

## Desarrollo

```bash
npm install --verbose
npm run dev
```

## Build de produccion

```bash
npx vite build
```

## Nota

Las imagenes son referenciales del rubro industrial/minero y pueden reemplazarse por fotografias reales de proyectos de la empresa para elevar conversion comercial.
